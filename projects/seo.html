SEO Audit & Web Performance Dashboard

Power BI Screaming Frog 10K+ URLs 21% crawl waste

Built a Power BI dashboard from crawl data to track website health KPIs, identify crawl budget waste, and prioritise technical/content fixes to improve search visibility.

Problem

Website issues were reducing visibility and efficiency, but priority fixes were unclear without structured reporting.

Approach

  • Imported Screaming Frog crawl export and cleaned fields for analysis.
  • Created KPI views for indexability, metadata quality, content depth, and performance signals.
  • Built summary + drill-down pages to support prioritisation and action planning.

Key Insights

  • Identified crawl budget waste and key indexability gaps affecting discoverability.
  • Flagged thin/duplicate content and missing metadata issues for fast wins.
  • Highlighted deep-page visibility limits impacting important pages.
